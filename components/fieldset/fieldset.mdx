## Wanneer gebruik je de Fieldset component

Gebruik de Form field component voor een representatie van een input element en de bijbehorende label en description.

## Wanneer gebruik je de Fieldset component niet

Gebruik de Form Field component niet om meerdere aan elkaar gerelateerde input elementen bij elkaar te zetten. Hiervoor is het Fieldset bedoelt.

### Fieldset met error status

Het is mogelijk om binnen de Fieldset component een visuele error status te tonen. De visuele vormgeving bestaat uit twee delen:

- Een border aan de linkerkant van de gehele Fieldset door gebruik van de `dictu-fieldset--error` class op het component.
- De combinatie van een icon en verklarende tekst boven het input element door middel van een extra `div` met de `"dictu-fieldset__error-message` class.
