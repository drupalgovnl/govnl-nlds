## Wanneer gebruik je de Textareacomponent

Gebruik de Textarea component wanneer de gebruiker tekst moet invoeren dat langer is dan één regel, zoals bijvoorbeeld een beschrijving of uitleg.

## Wanneer gebruik je de Text Input component niet

Gebruik de Textarea component niet wanneer het verwachte antwoord van de gebruiker bestaat uit tekst op slechts één regel. Gebruik dan de Text Input component.

### Textarea met placeholder

Het is mogelijk om een placeholder te laten zien in de textarea. Voeg hiervoor het `placeholder` attribuut toe aan het `textarea` element.

**Let op:** Een placeholder is nooit een vervanging voor het Label elemement. Gebruik alleen een placeholder als de informatie niet op een andere manier
weergegeven kan worden. Een placeholder verdwijnt namelijk wanneer de gebruiker het inputveld vult. Hierdoor is het niet mogelijk om de informatie uit
de placeholder tussentijds nogmaals te bekijken. Ook wanneer er een foutmelding wordt weergegeven, is het voor gebruikers niet mogelijk om te controleren
wat er precies fout is.
