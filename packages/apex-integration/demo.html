APEX Integration Architecture

Three-Layer Architecture

  1. Design Token Bridge Layer
    Maps DICTU CSS variables to APEX Universal Theme variables
    --a-cv-primary-color: var(--govnl-color-hemelblauw-500);
  2. Component Layer
    16 DICTU components with APEX-specific adaptations
  3. Template Adapter Layer
    APEX templates, plugins, and PL/SQL API

  • Installation scripts and documentation

Ready to Install?

Follow these steps to integrate DICTU Design System with your APEX application:

  1. Build the package: npm run build
  2. Upload CSS to APEX Static Files
  3. Run installation script: @sql/installation/install_dictu.sql
  4. Start using DICTU components!
