## Wanneer gebruik je de Link as Button component

Gebruik de Link as Button component wanneer de gebruiker naar een andere pagina navigeert, maar de link visueel moet opvallen als een call-to-action.

## Wanneer gebruik je de Link as Button component niet

Gebruik de Link as Button niet wanneer:

- De link in lopende tekst staat
- Er een actie uitgevoetrd moet worden, zoals bijvoorbeeld het versturen van een formulier, het openen van een modal of er iets op de pagina verandert zonder navigatie. Gebruik daarvoor het standaard [Button component](?path=/docs/componenten-button--docs).

## Voorbeelden

### Link as Button

#### Link as Button in primary action variant

Voor de primary action variant van de Link as Button voeg je de classes `.dictu-link-as-button` en `.dictu-link-as-button--primary-action` toe aan een `a` element.

<Canvas of={LinkAsButtonStories.DefaultLinkAsButton} />

#### Link as Button in secondary action variant

Naast de primary action variant kent de Link as Button ook een secondary variant. Voor deze variant voeg je de classes `.dictu-link-as-button` en `.dictu-link-as-button--secondary-action` toe aan het `a` element.
