## Wanneer gebruik je de Checkbox component

Gebruik de Radio Button component voor een keuzemogelijkheid waarbij slechts één optie gekozen kan worden.

## Wanneer gebruik je de Checkbox component niet

Gebruik de Radio Button component niet wanneer de gebruiker uit een lijst met keuzes meerdere opties mag selecteren, of wanneer er slechts één keuze is, zoals bij het akkoord
gaan met algemene voorwaarden. Hiervoor is het [Checkbox component](?path=/docs/componenten-form-elements-checkbox--docs) bedoelt.

## Richtlijnen

- Gebruik de Radio Button component altijd in combinatie met de [Label component](?path=/docs/componenten-form-elements-label--docs)
- Radio buttons worden altijd gebruikt in een groep van radio buttons waarin slechts één radio button de checked status mag hebben. Gebruik radio buttons daarom altijd in een
  [Fieldset component](?path=/docs/componenten-form-elements-fieldset--docs).
