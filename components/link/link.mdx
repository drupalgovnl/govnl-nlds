## Wanneer gebruik je de Link component

- Gebruik de Link component om te navigeren naar een interne of externe pagina, of om naar bronnen of bestanden te navigeren.

## Wanneer gebruik je de Link component niet

- Gebruik de Link component niet voor interactie op dezelfde pagina. Hiervoor wordt de Button component gebruikt.

### Link met icoon

Het is mogelijk om een icoon in de link te zetten. Voeg hiervoor de class `dictu-link--with-icon` aan het component toe. Gebruik de Icon component en geef als extra class `dictu-link__icon` mee.

<Canvas of={LinkStories.IconLink} />

### Externe link

Voor een externe link voeg je het icoon toe voor een external link. Geef een `label` aan de Icon component mee dat duidelijk maakt dat het om een externe link gaat.

<Canvas of={LinkStories.ExternalLink} />
