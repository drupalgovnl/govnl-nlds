## Wanneer gebruik je de Page Container component

- Gebruik de Page Section component om elementen samen te groeperen binnen een container.

## Richtlijnen

- Gebruik altijd de `.dictu-page-container` class voor de container-element.
- Gebruik semantische HTML: `<div>` of `<main>` voor de hoofdcontainer.
- Voeg indien nodig een `id` attribuut toe voor navigatie en toegankelijkheid.
- De component zorgt voor consistente maximale breedte en centrering van
  content.

### Page Container met gap tussen de elementen

Het is mogelijk om tussen de elementen in een page container witruimte toe te voegen. Er zijn drie mogelijke formaten witruimte: Small, medium, en large.
Voeg respectievelijk de class `dictu-container--gap-small`, `dictu-container--gap-medium` of `dictu-container--gap-large` toe.
