## Wanneer gebruik je de Button component

Gebruik de button component om gebruikers te helpen bij het uitvoeren van een actie die geen nieuwe pagina opent. Voorbeelden hiervan zijn: 

- Een formulier verzenden.
- Een modal openen/sluiten.
- Een filter activeren.
- Een dropdown openen.
- Een interactie binnen een single page application.

> Een vuistregel hierbij is: **"Buttons doen iets"**

## Wanneer gebruik je de Button component niet

Gebruik de button component niet als navigatie. Wanneer de gebruiker ergens naartoe gaat, gebruik dan altijd een `<a>`-link (anchor element). Voorbeelden wanneer je een link gebruikt in plaats van een button zijn:

- Gaan naar een andere pagina.
- Downloaden van een document.
- Linken naar externe websites.

## Richtlijnen

- Gebruik de `<button>` altijd voor het uitvoeren van een actie en niet als navigatie.
- Gebruik de `<button>`-tag om een klikbare knop te definiëren.
- Binnen het `<button>`-element kun je tekst plaatsen (en tags zoals `<i>`, `<b>`, `<strong>`, `<image>`, enz.).
- Geef altijd het type-attribuut op voor `<button>`-element, zodat browsers weten welk type knop het is.
- Zorg ervoor dat het klikbare element een minimale afmeting hebben van 24 bij 24 CSS-pixels.

## Voorbeelden

### Primary Action Button

De Primary Action Button wordt gebruikt om de meest belangrijke actie op de pagina te tonen. In de meeste gevallen wordt het aantal primaire acties (ofwel de Primary Action Button) beperkt tot één. Voor de andere acties kan de Secundary Action Button of de Subtle Button gebruikt worden.

<Canvas of={ButtonStories.PrimaryAction} />

### Secundary Action Button

De Secondary Action Button wordt voor minder prominente acties gebruikt. Het is mogelijk om meer dan één Secondary Action Button te gebruiken. De knop is visueel minder aanwezig dan de Primary Action Button.

<Canvas of={ButtonStories.SecondaryAction} />

### Subtle Button

De Subtle Button kan gebruikt worden voor de minst belangrijke acties op de pagina ten op zichte van de Primary Action Button en de Secondary Action Button. 

<Canvas of={ButtonStories.Subtle} />

### Icon Only Button

De Icon Only Button heeft een eenvoudig en herkenbaar icon die een specifieke actie of functionaliteit weergeeft. Ze worden gebruikt op plekken waar de ruimte beperkt is, zoals in toolbars, navigation. Bij voorkeur wordt er altijd gekeken of er een button met tekst gebruikt kan worden in verband met de toegankelijkheid

### Variabelen

#### Icon

De Primary Action Button, Secondary Action Button en de Subtle Button kunnen een icon bevatten. Standaard bevat de button component geen icon, de `iconPosition` krijgt in dit geval de waarde `none` mee. De button component kan een icon voor de tekst bevatten of na de tekst. Wanneer de icon voor de tekst staat krijgt de `iconPosition` de waarde `before` mee en na de tekst de waarde `after`.
