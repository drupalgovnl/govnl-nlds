## Wanneer gebruik je de Text Input component

Gebruik de Text Input component wanneer de gebruiker tekst moet invoeren dat niet langer is dan één regel, zoals bijvoorbeeld een straatnaam of telefoonnummer.

## Wanneer gebruik je de Text Input component niet

Gebruik de Text Input component niet wanneer het verwachte antwoord van de gebruiker bestaat uit tekst dat over meerdere regels valt. Gebruik dan de Textarea component.

## Richtlijnen

- Gebruik de Text Input component in combinatie met de [Label component](?path=/docs/componenten-form-elements-label--docs)

<Canvas of={TextInputStories.TextInputError} />

### Text input met placeholder

Het is mogelijk om een placeholder te laten zien in het inputveld. Voeg hiervoor het `placeholder` attribuut toe aan het `input` element.

**Let op:** Een placeholder is nooit een vervanging voor het Label elemement. Gebruik alleen een placeholder als de informatie niet op een andere manier 
weergegeven kan worden. Een placeholder verdwijnt namelijk wanneer de gebruiker het inputveld vult. Hierdoor is het niet mogelijk om de informatie uit
de placeholder tussentijds nogmaals te bekijken. Ook wanneer er een foutmelding wordt weergegeven, is het voor gebruikers niet mogelijk om te controleren
wat er precies fout is.
