## De Radio Button component implementeren

Je kunt de CSS zo in je project installeren:

```console
npm install --save-dev @dictu/radio-button
```

Je kunt de CSS uit `node_modules/` importeren:

```html
<link rel="stylesheet" href="node_modules/@dictu/radio-button/dist/index.css" />
```

Als je CSS imports gebruikt vanuit JavaScript:

```javascript
import "@dictu/radio-button/dist/index.css";
```
