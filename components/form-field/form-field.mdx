## Wanneer gebruik je de Form field component

Gebruik de Form field component voor een representatie van een input element en de bijbehorende label en description.

## Wanneer gebruik je de Form field component niet

Gebruik de Form Field component niet om meerdere aan elkaar gerelateerde input elementen bij elkaar te zetten. Hiervoor is het Fieldset bedoelt.

### Form field met error status

Het is mogelijk om binnen de Form field component een visuele error status te tonen. De visuele vormgeving bestaat uit twee delen:

- Een border aan de linkerkant van de gehele Form field door gebruik van de `dictu-form-field--error` class op het component.
- De combinatie van een icon en verklarende tekst boven het input element door middel van een extra `div` met de `"dictu-form-field__error-message` class.

<Canvas of={FormFieldStories.ErrorFormField} />

### Form field met dikgedrukte label

Doordat de Form field component gebruikt maakt van het Label component, kunnen alle Label varianten gebruikt worden.

<Canvas of={FormFieldStories.LabelStrongFormField} />

### Inline Form field

Bij bepaalde input types wordt de label niet boven, maar achter de input gezet. Dit is bijvoorbeeld het geval bij checkbox en radio elementen. Voor deze input types kan de inline Form field gebruikt worden.

Voor de inline Form field wordt de `dictu-form-field--inline` class toegevoegd aan het component, dat ervoor zorgt dat de elementen niet onder elkaar, maar achter elkaar komen te staan.

Bij de inline variant is het niet mogelijk om een description of error status te gebruiken.
